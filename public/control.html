⚙ How to set it up live

  1. Create a free Twilio account, buy a number (~$1/mo).
  2. Set the number's Messaging webhook to
  3. On the client's phone, forward missed calls to that Twilio number.
  4. Add TWILIO_SID, TWILIO_TOKEN, TWILIO_FROM to .env.

  3. Verify token: frontdesk-verify (or set IG_VERIFY_TOKEN).
