Missed-call → text

Caller hits voicemail → gets an instant text back.

⚙ How to set it up live
  1. Create a free Twilio account, buy a number (~$1/mo).
  2. In Twilio, set the number's Messaging webhook to: …/api/CLIENT/sms
  3. On the client's phone, turn on "forward missed calls" to that Twilio number.
  4. Add TWILIO_SID, TWILIO_TOKEN, TWILIO_FROM to your .env.
  5. Now every missed call gets an instant AI text back. (Until then it's simulated above.)

Instagram DM

DM auto-reply with a booking link.

⚙ How to set it up live
  1. Client connects their Instagram to a Meta Business account.
  2. In Meta Developer, add a webhook pointing to: …/api/CLIENT/instagram
  3. Verify token: frontdesk-verify (or set IG_VERIFY_TOKEN).
  4. Add IG_TOKEN and IG_PAGE_ID to your .env.
  5. Now every DM auto-replies with an answer + booking link. (Until then it's simulated above.)
